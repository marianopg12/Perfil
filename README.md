# Player Profile Dashboard

Aplicación de Streamlit para visualizar perfiles de jugadores con métricas de rendimiento físico.

## 🚀 Instalación

```bash
pip install -r requirements.txt
```

## ▶️ Ejecutar

```bash
streamlit run perfil.py
python -m streamlit run perfil.py
```

## 📊 Datos requeridos

Archivo Excel `jugadores_stats.xls` con hojas:
- Antropometria
- Fuerza
- Saltabilidad
- Tracking GPS
